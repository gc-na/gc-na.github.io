🔍 효율적인 검색 팁

1. Google 뿐만 아니라, Semantic Scholar | Chat GPT | Perplexity도 사용하십시오.

2. 검색어를 잘 주셔야합니다. 

인터넷에서 원하는 자료를 빠르게 찾으려면 검색어 선택이 중요합니다. 같은 단어라도 맥락에 따라 검색 결과가 크게 달라지기 때문입니다.

​

​

예를 들어:

"SoC"라고만 검색하면 일반적인 모든 SoC 자료가 나옵니다.

![1](/assets/img/223711981478/1.png)

"SoC in VLSI"라고 제한하면, VLSI에 관련된 SoC 자료만 나옵니다. [참고로, in VLSI라는 키워드를 넣으면... 정말 놀랍게도 반도체를 쉽게 찾습니다! Design ]

![2](/assets/img/223711981478/2.png)

​

아래는 제가 자료를 빠르게 찾는 주요 방법과 팁입니다.

​

1. 데이터베이스 선택

반적인 웹 검색 대신 신뢰할 수 있는 전문 데이터베이스를 활용합니다.

논문/학술 자료: Google Scholar, PubMed, IEEE Xplore, Springer, Elsevier

특허 검색: Google Patents, USPTO, WIPO

기술 자료: 기업 공식 웹사이트, Accellera, Synopsys, ARM Documentation

팁:

학술 자료는 Google Scholar에서 특정 키워드와 연도를 설정해 검색하면 신속히 최신 정보를 확인할 수 있습니다.

특허 검색에서는 키워드 외에 특허 번호, 등록 국가를 활용하면 효율적입니다.

​

2. 검색 연산자 사용

RA 컨설턴트들은 단순한 키워드 검색 대신, 고급 검색 연산자를 활용해 검색 결과를 세밀하게 필터링합니다.

주요 연산자

"" (정확한 문구 검색): "VLSI Physical Design"

site: (특정 사이트 내 검색): site:ieee.org

filetype: (특정 파일 형식 검색): filetype:pdf

intitle: (제목에 포함된 검색어): intitle:"Semiconductor Trends"

- (제외 검색어): -제외단어

예시:

"VLSI" site:springer.com filetype:pdf

→ Springer에서 제공하는 RISC-V 관련 PDF 파일만 검색

​

3. 키워드 확장 및 세분화

단순 키워드 대신 분야별 관련어를 활용하여 검색합니다.

기본 키워드: Verilog

확장 키워드: Verilog Syntax, Verilog HDL Basics, Verilog for RTL Design

​

4. 검색 필터 활용

검색 엔진과 데이터베이스에서 제공하는 필터 옵션을 적극 활용합니다.

출판 연도: 최신 자료를 우선시 (예: 최근 5년)

카테고리: 논문, 튜토리얼, 기술 보고서 등 선택

언어 필터: 영어 자료가 더 풍부한 경우가 많음

​

5. AI 도구 활용

RA 컨설턴트들은 AI 기반 검색 도구를 활용하여 자료를 보다 효율적으로 찾습니다.

Semantic Scholar: 논문과 학술 자료를 문맥 기반으로 검색

Perplexity AI: 간단한 질의어로 전문 자료 검색

ChatGPT와 같은 AI: 복잡한 질문에 대한 요약 자료 및 링크 제공

팁:

> AI 검색은 단순 검색 이상의 맥락을 파악하여 결과를 제공하므로,질문을 구체적으로 작성하세요. AI는 검색엔진이 아닙니다.예: "Explain the difference between ARM Cortex-M and Cortex-A"

출처 : No Site

​

6. 주제별 커뮤니티와 포럼 활용

키워드 조차 잘 떠오르지 않는다면, 사람들에게 물어보십시오.

반도체 포럼: Linkedin, SemiWiki, EDA Board, Reddit 일부 게시판 ...

​

7. 논리적인 검색 흐름":

문제 정의: "어떤 자료가 필요한가?"

예: VLSI 설계 흐름 이해 → Verilog 튜토리얼 검색

키워드 리스트 작성: 관련된 주요 키워드와 연관된 단어 나열

단계별 검색: 생각나는 대로 키워드 넣고 검색 → 검색 연산자(예를들어 "vlsi"라고 검색어에 포함하면, vlsi 문자열이 반드시 포함된 결과만 출력됨.) 사용 → 검색 필터 조정(어떤 기간, 어떤 언어로 된 자료를 출력할 것인지)

8. 신뢰할 수 있는 자료 검증

출처가 명확한 공식 문서나 학술 자료를 우선적으로 사용

데이터가 오래되었거나 검증되지 않은 자료는 제외

​

효율적인 검색 요약

구체적인 키워드로 검색 범위를 좁힌다.

전문 데이터베이스를 사용한다.

검색 연산자를 활용한다.

검색 필터 옵션을 사용한다.

검증된 + 최신 자료가 가장 질 좋은 출력물.
